# Vungeon - 3D 미로 탈출 게임

## 📌 개요

**Vungeon**은 Three.js를 활용하여 개발한 3D 미로 탈출 게임입니다. 플레이어는 다양한 던전 방들을 탐험하며 보물 방을 찾아 탈출해야 합니다. 게임을 클리어할수록 미로의 크기가 증가하여 난이도가 높아집니다.

## ⚙️ 주요 기능

- **동적 미로 생성**: Recursive Backtracking 알고리즘으로 매 게임마다 랜덤 미로 생성
- **동적 방 로딩**: 플레이어 주변의 방만 로딩하여 성능 최적화
- **8가지 던전 방 타입**: 미로 구조에 맞는 다양한 방 자동 배치
- **충돌 감지**: AABB 기반 정밀한 충돌 처리
- **난이도 시스템**: 클리어할수록 미로 크기 증가 (10x10 → 최대 30x30)
- **타이머**: 실시간 플레이 타임 기록 및 결과 표시
- **포스트 프로세싱**: Bloom 효과를 활용한 시각 효과

## 🛠 기술 스택

- **Three.js 0.158.0**: 3D 렌더링 엔진
- **JavaScript (ES6 Modules)**: 모듈화된 코드 구조
- **HTML5 / CSS3**: UI 및 스타일링
- **OBJ/MTL Loader**: 3D 모델 로딩
- **EffectComposer**: 포스트 프로세싱 (Bloom 효과)

## 🚀 실행 방법

1. 프로젝트 폴더에서 Python HTTP 서버 실행:
   ```bash
   python -m http.server 8000
   ```

2. 브라우저에서 접속:
   ```
   http://localhost:8000
   ```

## 🎯 조작 방법

- **W**: 위쪽으로 이동
- **S**: 아래쪽으로 이동
- **A**: 왼쪽으로 이동
- **D**: 오른쪽으로 이동

## ⚠️ 기타 참고사항

- **브라우저**: Chrome, Edge, Firefox 최신 버전 권장 (ES6 Modules, WebGL 지원 필요)
- **성능**: 저사양 PC에서는 프레임 드롭이 발생할 수 있음
- **게임 특징**: 미로는 매 게임마다 랜덤 생성, 클리어 시 자동으로 다음 레벨 시작
